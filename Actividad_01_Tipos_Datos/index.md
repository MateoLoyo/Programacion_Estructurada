# Actividad 01: Tipos de Datos

## 📋 Descripción

Programa que declara, inicializa e imprime los principales tipos de datos primitivos del lenguaje C, mostrando su dirección de memoria y valor asignado.

## 🎯 Temas aplicados

- Tipos de datos
- printf
- Direcciones de memoria
