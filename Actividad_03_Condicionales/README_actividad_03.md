# Actividad: Condicionales

## Descripción

Programas que demuestran el uso de estructuras condicionales (if, if-else, if-else if) y operadores relacionales/lógicos para la toma de decisiones en lenguaje C.

## Temas aplicados

- condicionales
- if-else
- operadores relacionales
- operadores logicos
- scanf
- printf

## Compilación

- gcc Indicar_Par.c -o indicar_par
- gcc Indicar_Impar.c -o indicar_impar
- gcc Indicar_Primo.c -o indicar_primo
- gcc Mayor_Edad.c -o mayor_edad
- gcc Promedio_MateriasR.c -o promedio_materias
- gcc Sistema_Validacion.c -o sistema_validacion

## Ejecución

- ./indicar_par
- ./indicar_impar
- ./indicar_primo
- ./mayor_edad
- ./promedio_materias
- ./sistema_validacion
