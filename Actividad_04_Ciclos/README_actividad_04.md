# Actividad: Ciclos

## Descripción

Programas que demuestran el uso de estructuras repetitivas (for, while, do-while) y su aplicación en la resolución de problemas en lenguaje C.

## Temas aplicados

- ciclos
- for
- while
- do-while
- acumuladores
- contadores
- validación de datos

## Compilación

- gcc For.c -o for
- gcc While.c -o while
- gcc DoWhile.c -o dowhile
- gcc Calculadora_Promedios.c -o calculadora_promedios

## Ejecución

- ./for
- ./while
- ./dowhile
- ./calculadora_promedios
