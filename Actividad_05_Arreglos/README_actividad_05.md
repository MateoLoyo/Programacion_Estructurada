# Actividad: Arreglos

## Descripción

Programas que demuestran la declaración, inicialización, acceso y manejo de arreglos unidimensionales en lenguaje C, incluyendo el análisis de direcciones de memoria y el uso de arreglos de tamaño dinámico.

## Temas aplicados

- arreglos
- ciclos
- direcciones de memoria
- scanf
- printf

## Compilación

- gcc Arreglos.c -o arreglos
- gcc Arreglos_2.c -o arreglos_2

## Ejecución

- ./arreglos
- ./arreglos_2
