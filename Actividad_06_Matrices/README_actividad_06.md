# Actividad: Matrices

## Descripción

Programas que demuestran la declaración, inicialización, acceso y manejo de matrices bidimensionales en lenguaje C, incluyendo el uso de matrices de caracteres, análisis de direcciones de memoria y manejo de archivos con matrices.

## Temas aplicados

- matrices
- ciclos anidados
- direcciones de memoria
- manejo de archivos
- cadenas de caracteres

## Compilación

- gcc Matriz.c -o matriz
- gcc Matriz_Nombre_Completo.c -o matriz_nombre
- gcc File_Martriz.c -o file_matriz

## Ejecución

- ./matriz
- ./matriz_nombre
- ./file_matriz
