# Actividad: Cadenas

## Descripción

Programa interactivo con menú que demuestra el manejo de cadenas de caracteres en lenguaje C, incluyendo lectura con y sin espacios, uso de funciones de la librería string.h y arreglos de cadenas.

## Temas aplicados

- cadenas
- scanf
- fgets
- strlen
- strcat
- strcmp
- strcpy
- arreglos de cadenas
- funciones
- switch

## Compilación

- gcc string.c -o string

## Ejecución

- ./string
