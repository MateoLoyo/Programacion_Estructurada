# Actividad: Funciones

## Descripción

Programas que demuestran la declaración, definición y llamada de funciones en lenguaje C, incluyendo paso de parámetros por valor, funciones con retorno, funciones void, y funciones que operan con arreglos y archivos.

## Temas aplicados

- funciones
- paso por valor
- paso de arreglos
- funciones con retorno
- funciones void
- manejo de archivos
- filtrado de datos

## Compilación

- gcc Funciones.c -o funciones
- gcc Funciones+Archivos.c -o funciones_archivos

## Ejecución

- ./funciones
- ./funciones_archivos
