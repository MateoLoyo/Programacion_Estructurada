# Actividad: Punteros

## Descripción

Programas que demuestran el uso de punteros en lenguaje C, incluyendo declaración e inicialización, dereferencia, modificación indirecta de variables, paso de parámetros por referencia a funciones, y la relación entre arreglos y punteros.

## Temas aplicados

- punteros
- direcciones de memoria
- dereferencia (*)
- operador de dirección (&)
- paso por referencia
- arreglos y punteros
- aritmética de punteros
- funciones con punteros

## Compilación

- gcc Punteros.c -o punteros
- gcc Paso_Referencia.c -o paso_referencia
- gcc Arreglos_Punteros.c -o arreglos_punteros

## Ejecución

- ./punteros
- ./paso_referencia
- ./arreglos_punteros
