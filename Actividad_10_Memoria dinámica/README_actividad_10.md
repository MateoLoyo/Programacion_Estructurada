# Actividad: Memoria Dinámica

## Descripción

Programas que demuestran la reserva y liberación de memoria dinámica en lenguaje C usando malloc, calloc, realloc y free, basados estrictamente en las guías de estudio.

## Temas aplicados

- malloc
- calloc
- realloc
- free
- memoria dinámica
- sizeof
- stdlib.h

## Compilación

- gcc Malloc_Basico.c -o malloc_basico
- gcc Calloc_Inicializa.c -o calloc_inicializa
- gcc Realloc_Cambiar.c -o realloc_cambiar

## Ejecución

- ./malloc_basico
- ./calloc_inicializa
- ./realloc_cambiar
