# Actividad: Struct y Union

## Descripción

Programas que demuestran el uso de struct para agrupar variables de distinto tipo y union para compartir memoria entre variables.

## Temas aplicados

- struct
- union
- arreglos de struct
- acceso con punto (.)
- fgets
- scanf
- printf

## Compilación

- gcc Struct_Alumno.c -o struct_alumno
- gcc Struct_Arreglo.c -o struct_arreglo
- gcc Union_Dato.c -o union_dato

## Ejecución

- ./struct_alumno
- ./struct_arreglo
- ./union_dato
