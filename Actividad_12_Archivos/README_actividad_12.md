# Actividad: Archivos

## Descripción

Programas que demuestran el manejo de archivos en lenguaje C, incluyendo escritura, lectura y combinación con matrices y funciones.

## Temas aplicados

- fopen
- fclose
- fprintf
- fgets
- modo escritura ("w")
- modo lectura ("r")
- archivos de texto
- matrices de caracteres
- funciones
- arreglos
- filtrado de datos
- strcspn
- fgets

## Compilación

- gcc File_Martriz.c -o file_matriz
- gcc Funciones+Archivos.c -o funciones_archivos

## Ejecución

- ./file_matriz
- ./funciones_archivos
